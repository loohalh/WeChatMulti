# 微信多开启动器

配合[WeChatTweak](https://github.com/loohalh/WeChatTweak)使用，才有效果

## 使用
1 、安装微信，安装[WeChatTweak](https://github.com/loohalh/WeChatTweak)

2、安装WeChatMulti

3、应用程序点开Wechat(蓝色图标)即可多开
